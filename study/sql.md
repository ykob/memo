# SQL

## データ型

### 数値型

- `int` ：整数
- `decimal` ：固定長桁数、固定長少数桁数を持つ数値
- `money` ：通貨
- `float` ：浮動小数点数値

### 文字列型

- `char` ：8000文字以内の固定長の文字列
- `nchar` ：4000文字以内の固定長の文字列(UNICODE)
- `varchar` ：8000文字以内の可変長の文字列
- `nvarchar` ：4000文字以内の可変長の文字列(UNICODE)
- `text` ：20億文字以内の可変長の文字列
- `ntext` ：10億文字以内の可変長の文字列(UNICODE)

### 日付型

- `datetime` ：日付
