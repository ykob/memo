## 第1章 AWSを使う

- クラウドコンピューティングのメガベンダー
  - Amazon (AWS)
  - Microsoft (Azure)
  - Google (Google Cloud Platform)
- AWSのユーザーが必要なITリソースを選択して使用する
- AWSのクラウド製品概要
  - コンピューティング（サーバー）... EC2, ECS
  - ストレージ ... S3, EBS
  - データベース ... RDS, DynamoDB
  - ネットワーク ... VPC
  - エンドユーザーコンピューティング(クライアント) ... WorkSpaces, WorkLink
- 複雑な最新技術を比較的容易に利用できる
- Cloud Watch ... AWSで利用するITリソースの運用監視を行うサービス
